Basic usage

You can add a wide variety of default animations to your objects just by calling .animatio method. You only need to set the type of animation you want to run among a wide variety of preset animations (see below).

<script type="text/javascript">

  $(selector).animatio(type);

</script>

Configure your animation

Animations can be highly set up by adding a second param as a dictionary. Those settings are:

  • duration Default: '1s'
  • timing Default: 'ease'
  • plays Default: 1
  • direction Default: 'normal'
  • delay Default: '100ms'
  • fill Default: 'forwards'
You can find more info about those configuration parameters at the official w3c document 

<script type="text/javascript">

  $(selector).animatio('pulse', {
    delay: '1s',
    direction: 'normal',
    duration:'500ms',
    fill: 'forwards',
    plays: 'infinite',
    timing: 'ease'
  });

</script>

Define a completion callback

It's possible to add an optional third parameter to define a callback function that will be executed when the animation ends.

<script type="text/javascript">

  $(selector).animatio('fadeIn', {
    // configuration
  },
  function(){
    // execute now
    $(this).animatio('fadeOut');
  });

</script>

Create your own animation

There are predefined animations that are ready to use (listed below). However, you can also create your own animation by setting it as a string in config.rule.

{browser} is a place-holder that we will be replaced with the browser prefix when the rule is actually created.

<script type="text/javascript">

  $(selector).animatio('move_a_little', {
    rule: "0% { {browser}transform: translateX(0);}, 100% { {browser}transform: translateX(100);}"
  });

</script>
